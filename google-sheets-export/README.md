# TMK — ส่งออกข้อมูลไป Google Sheet (อ่านอย่างเดียว)

ดึงข้อมูลจาก **Supabase → Google Sheet** เพื่อ "ดู" อย่างเดียว
กรอกข้อมูลจริงทำในเว็บเหมือนเดิม — ชีตนี้ไม่เขียนกลับ ไม่กระทบฐานข้อมูลเลย (ยิงแค่ GET)

## โครงชีตที่ได้
- **`⚙️ ตั้งค่า`** — กรอกช่วงวันที่ที่อยากดึง + ดูเวลาดึงล่าสุด
- **`📊 สรุปรายเดือน`** — ทุกเดือน 1 แถว (เป้า/ยอดจริง/%เป้า/Run Rate/ออเดอร์/AOV/ค่าแอด/ACOS/ROAS/ลูกค้า/CAC/ข้อความ)
- **`YYYY-MM ...`** — แต่ละเดือน 1 แท็บ — **1 แถว = 1 วัน** แยกแต่ละช่องเป็นคอลัมน์ (ยอด/ออเดอร์/แอด/ใหม่/เก่า) + ยอดรวมทุกช่อง สร้างเองอัตโนมัติ

## ติดตั้งครั้งเดียว (3 นาที)
1. เปิด [sheets.new](https://sheets.new) สร้าง Google Sheet เปล่า 1 ไฟล์
2. เมนู **ส่วนขยาย (Extensions) → Apps Script**
3. ลบโค้ดตัวอย่างทิ้ง → วางเนื้อหาทั้งหมดจากไฟล์ [`Code.gs`](./Code.gs) → กด 💾 บันทึก
4. กลับมาที่ชีต **รีโหลดหน้า 1 ครั้ง** → จะมีเมนูใหม่ชื่อ **`TMK`** บนแถบเมนู
5. กด **`TMK ▸ ⟳ ดึงข้อมูลล่าสุด`** → ครั้งแรก Google จะขออนุญาต → กด **Review permissions → เลือกบัญชี → Advanced → Go to (ชื่อโปรเจกต์) → Allow**
6. รออึดใจ → แท็บเดือน + สรุปโผล่ครบ ✓

> การขออนุญาตครั้งแรกเป็นเรื่องปกติของ Apps Script (เพราะสคริปต์ขอสิทธิ์ "ต่อเน็ตออกไป Supabase" + "เขียนลงชีตของคุณเอง") — ไม่เกี่ยวกับสิทธิ์ในเว็บ/ฐานข้อมูล

## ใช้งานประจำ
- อยากดูข้อมูลล่าสุด → กด **`TMK ▸ ⟳ ดึงข้อมูลล่าสุด`** (กดกี่ครั้งก็ได้ ปลอดภัย)
- อยากดูเฉพาะช่วง → ไปแท็บ **`⚙️ ตั้งค่า`** กรอก **วันที่เริ่ม / วันที่สิ้นสุด** (รูปแบบ `2026-01-01`) แล้วกดดึงใหม่
- เว้นช่องวันที่ว่าง = ดึงทั้งหมด

## ปลอดภัย & รองรับระยะยาว
- **อ่านอย่างเดียว** — สคริปต์มีแต่คำสั่ง GET เปลี่ยน/ลบข้อมูลใน Supabase ไม่ได้
- **กันข้อมูลเดิมหาย** — ถ้าเน็ตหลุดตอนดึง จะแจ้งเตือนและไม่แตะแท็บเดิม
- **เผื่ออนาคตหลายปี** — แต่ละเดือนแยกแท็บ (~180 แถว/เดือน) เปิดเร็วตลอด · ดึงทีละ 1,000 แถวอัตโนมัติ · ช่องทางที่เพิ่มในเว็บโผล่ในชีตเอง

## ปรับแต่ง (ถ้าต้องการ)
ในไฟล์ `Code.gs` ด้านบนสุด:
- `SUPABASE_URL` / `SUPABASE_ANON_KEY` — ใส่ไว้ให้แล้ว (anon key เป็น public เหมือนที่ติดมากับเว็บ)
- ถ้าเปลี่ยนโปรเจกต์ Supabase ใหม่ แก้ 2 ค่านี้

## แก้ปัญหาเบื้องต้น
| อาการ | แก้ |
|---|---|
| ไม่เห็นเมนู TMK | รีโหลดหน้าชีต / เช็คว่า Save สคริปต์แล้ว |
| `HTTP 401/403` | anon key ไม่ถูก — ตรวจ `SUPABASE_ANON_KEY` |
| `ไม่พบช่องทาง` | ตาราง `tmk_channels` ว่าง/URL ผิด |
| ช้า/แดง timeout | กรอกช่วงวันที่ให้แคบลงในแท็บตั้งค่า |
